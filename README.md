# AI/ML Engineer Portfolio - Abhishek Kushwah

A modern, research-oriented, and highly professional portfolio website built with React, Tailwind CSS, and Framer Motion.

## 🚀 Tech Stack

- **Frontend**: React (Vite)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Design Aesthetic**: Inspired by OpenAI + Apple (Minimal, dark-themed, glassmorphism)

## 📁 Structure

- `src/components/common`: Reusable UI elements (Navbar, Footer, Section Wrapper).
- `src/components/sections`: Main page sections (Hero, About, Skills, Projects, Experience, Research, Contact).
- `src/data/content.ts`: Centralized data file for easy updates.
- `src/hooks`: Custom React hooks (Typing effect).
- `src/assets`: Images and profile photos.

## 🛠️ Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```
2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
3.  **Build for Production**:
    ```bash
    npm run build
    ```
